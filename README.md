# React Project Generator

### A fast AI automation to create a React Project, along with Tailwind and Supabase Database

___

# Dependencies

### File Strucutre (10 seconds):

- To python

` pip install google-generativeai `

### Credentials (4 minutes)

This IA use Google AI Studio models to program and a Supabase Account. The main file needs Supabase Project URL, Supabse Project KEY and API Key from Google

So save this informations:

- Supabase URL
- Supabase KEY
- Google AI Studio API Key

- Create an account: https://supabase.com

- Create a new project

- Find **SQL Editor** (Or press CTRL + E in your Project Overview)

- Paste this code:
```
create table app_universal (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  collection text not null,
  data jsonb not null
);
alter table app_universal enable row level security;
create policy "Liberar Geral" on app_universal for all using (true) with check (true);
```

### To get Supabase credentials:

In Your Project Settings, find:

And in API Keys you will find your Supabase Key (Something like sb_publishable[...] or sb_secret[...])
